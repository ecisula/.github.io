# 30-денний план просування і продажів (Rozetka + Prom + EpicentrK)

Адаптуй під свою нішу: **категорія товару, бюджет/місяць, кількість SKU**.

## День 1–3: Підготовка основи
- Зібрати SKU-матрицю: ТОП (20%), середні (50%), довгий хвіст (30%).
- Зафіксувати цілі: маржа, мін. ціна, цільовий CTR, цільова конверсія.
- Підготувати фід/інтеграції (ціни, залишки, статуси).
- KPI: готові 100% атрибути для стартових SKU.

## День 4–7: Запуск контенту
- Для кожного SKU: назва з ключами, повні характеристики, 6–10 фото, 1 відео (де можливо).
- Уніфікувати шаблон опису: вигода → характеристики → гарантія → доставка/повернення.
- Проставити правильні категорії/фільтри на кожному маркетплейсі.
- KPI: не менше 30–50 SKU опубліковано без помилок модерації.

## День 8–10: Ціноутворення і конкурентність
- Визначити 10–15 “локомотивів” з найкращою ціною.
- Для інших SKU: цінові сходинки + комплекти (bundle) для росту середнього чеку.
- Налаштувати автооновлення залишків/цін мінімум 2–4 рази на день.
- KPI: +CTR у ключових SKU, відсутність out-of-stock у топ-позиціях.

## День 11–14: Реклама всередині майданчиків
- Запустити рекламу на ТОП SKU + новинки.
- Окремо тести: 3 групи товарів (висока маржа / висока конверсія / сезонні).
- Щодня відключати збиткові оголошення, підсилювати прибуткові.
- KPI: стабільний ROAS/ACOS у межах твоєї маржі.

## День 15–18: Довіра і відгуки
- Автоматизувати запити на відгук після отримання замовлення.
- Додати в картки блоки: гарантія, обмін/повернення, терміни відправки.
- Відпрацювання негативу за SLA (макс. 24 год).
- KPI: приріст кількості відгуків і рейтингу магазину.

## День 19–22: Операційна швидкість
- Скоротити час обробки замовлення і відправки.
- Контроль відміни/повернень: причини, корекції карток і очікувань клієнта.
- Перевірити відповідність фото/опису фактичному товару.
- KPI: менше скасувань, кращий seller performance.

## День 23–26: Масштабування SKU
- Додати нові SKU тільки з категорій, де вже є продажі.
- Запустити A/B тест заголовків і головного фото.
- Розширити перехресні продажі: аксесуари/комплекти.
- KPI: ріст обороту без просідання маржі.

## День 27–30: Оптимізація і план на наступний місяць
- Аналіз по кожному SKU: покази → CTR → конверсія → чистий прибуток.
- Вимкнути “мертві” SKU, перезапакувати слабкі картки.
- Сформувати пул SKU для агресивного бусту на наступні 30 днів.
- KPI: фінальний P&L по каналах Rozetka/Prom/EpicentrK.

## Розподіл бюджету (базово)
- 60%: товари з доведеною конверсією.
- 25%: новинки/тести.
- 15%: ретест “просілих” SKU після покращення карток.
